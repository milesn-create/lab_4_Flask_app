## Веб-приложение для управления учетными записями пользователей (CRUD)

###  Описание проекта

Данное веб-приложение разработано с использованием фреймворка Flask и реализует полный CRUD-функционал для управления учетными записями пользователей.

Приложение позволяет:

* создавать пользователей
* просматривать информацию о пользователях
* редактировать данные пользователей
* удалять пользователей

Также реализована система аутентификации и смены пароля.

---

##  Технологии

* Python 3
* Flask
* SQLite
* HTML / Jinja2
* CSS (оформление интерфейса)

##  Функциональность

###  Аутентификация

* Вход в систему по логину и паролю
* Данные пользователей загружаются из базы данных
* Ограничение доступа к функциям (создание, редактирование, удаление)

---

###  Пользователи

Каждый пользователь содержит:

* ID
* логин
* хеш пароля
* имя
* фамилию (может отсутствовать)
* отчество (может отсутствовать)
* роль (может отсутствовать)
* дату создания (устанавливается автоматически)

---

###  Главная страница

* отображает список пользователей в виде таблицы
* содержит:

  * порядковый номер
  * ФИО
  * роль
  * действия (просмотр / редактирование / удаление)

 Кнопки "Редактировать" и "Удалить":

* доступны только авторизованным пользователям

 Кнопка "Создать пользователя":

* доступна только авторизованным пользователям

---

###  Просмотр пользователя

* отображение данных пользователя в виде таблицы
* доступно всем пользователям

---

###  Создание пользователя

* форма с полями:

  * логин
  * пароль
  * имя
  * фамилия
  * отчество
  * роль
* при ошибке:

  * выводится сообщение (flash)
  * форма сохраняет введённые данные
* при успехе:

  * редирект на главную страницу

---

###  Редактирование пользователя

* форма автоматически заполняется текущими данными
* отсутствуют поля:

  * логин
  * пароль
* используется общий шаблон формы (без дублирования)
* доступ только для авторизованных

---

###  Удаление пользователя

* реализовано через модальное окно
* подтверждение:

  ```
  "Вы уверены, что хотите удалить пользователя ФИО?"
  ```
* кнопки:

  * Да → удаление
  * Нет → отмена
* используется POST-запрос
* доступ только для авторизованных

---

###  Смена пароля

* форма содержит:

  * старый пароль
  * новый пароль
  * подтверждение пароля
* проверки:

  * корректность старого пароля
  * соответствие нового пароля требованиям
  * совпадение паролей
* при успехе:

  * редирект на главную
  * сообщение об успехе

---

##  Валидация данных

### Логин:

* только латинские буквы и цифры
* минимум 5 символов

### Пароль:

* от 8 до 128 символов
* минимум:

  * 1 заглавная буква
  * 1 строчная буква
  * 1 цифра
* без пробелов
* допустимые спецсимволы:

  ```
  ~ ! ? @ # $ % ^ & * _ - + ( ) [ ] { } > < / \ | " ' . , : ;
  ```

### Обязательные поля:

* логин
* пароль
* имя
* фамилия (по ТЗ — обязательна, если реализовано)

 При ошибке:

* подсветка поля
* сообщение под полем

---

##  Работа с базой данных

* используется SQLite
* база создаётся автоматически при запуске
* таблицы:

  * users
  * roles
* реализованы:

  * связи (FOREIGN KEY)
  * хеширование паролей

##  Запуск проекта

1. Установить зависимости:

```bash
pip install flask
```

2. Запустить приложение:

```bash
python app.py
```

3. Открыть в браузере:

```
http://localhost:5001
```
